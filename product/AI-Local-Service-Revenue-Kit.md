# AI Local Service Revenue Kit

This is a ready-to-sell digital product for local service businesses that lose revenue because calls, forms, quotes, and old leads do not get followed up quickly.

Suggested price: $29

Suggested buyer: owner-operators in home services, detailing, cleaning, landscaping, med spas, clinics, fitness studios, repair, tutoring, and other appointment-based local businesses.

## Quick Start

1. Pick one inbox where leads arrive: phone, website form, Facebook, Google Business Profile, Yelp, Thumbtack, or email.
2. Create four labels: `New Lead`, `Quoted`, `Needs Follow-Up`, `Won/Lost`.
3. Use the reply prompts below for every new lead within five minutes.
4. Run the daily follow-up SOP before lunch.
5. Ask every completed happy customer for a review the same day.

## Revenue Recovery Prompts

Replace bracketed text before sending.

### Missed Call Text-Back

Hi [Name], sorry we missed you. This is [Business]. What can we help with today? Send a few details and, if useful, a photo. We can usually give next steps quickly.

### Speedy Quote Reply

Thanks for reaching out. To quote this accurately, can you send:

- Location or zip code
- What you need done
- Preferred timing
- Any photos or measurements

Once I have that, I can give you a clear next step.

### Quote Follow-Up: Same Day

Hi [Name], quick check-in on the quote for [Job]. Do you want me to hold a spot for [Date/Window], or should I adjust the scope to fit a different budget?

### Quote Follow-Up: 48 Hours

Hi [Name], wanted to make sure my quote did not get buried. We can still help with [Job]. Want me to send two options: best value and fastest available?

### Quote Follow-Up: 7 Days

Hi [Name], should I close this out for now, or are you still considering help with [Job]? Either answer is totally fine.

### Price Objection

I understand. The price reflects [materials/time/licensing/quality factor]. If helpful, I can suggest a lighter version that keeps the most important part and lowers the total.

### Scheduling Push

We have [Time Window 1] or [Time Window 2] available. Which one works better?

### No-Show Recovery

Hi [Name], looks like we missed each other for [Appointment]. Do you want to reschedule, or should I close this request for now?

### Review Request

Thanks again for choosing us. If you were happy with the work, a quick Google review would mean a lot and helps local customers find us: [Review Link]

## AI Prompt Bank

Use these with ChatGPT, Claude, Gemini, or any AI writing tool.

### Lead Reply Generator

Act as a friendly customer service assistant for a local [business type]. Write a concise SMS reply to a new lead. Tone: warm, confident, direct. Ask only the minimum questions needed to qualify the job. Business details: [details]. Lead message: [paste lead].

### Quote Follow-Up Generator

Create three follow-up messages for a prospect who received a quote but has not booked. Write one same-day message, one 48-hour message, and one seven-day message. Keep each under 320 characters. Avoid pressure. Include a clear next step.

### Review Ask Generator

Write five review request messages for customers who just completed [service]. Tone should be grateful and simple. Include this review link: [link]. Avoid sounding desperate.

### Old Lead Reactivation

Write a short campaign to reactivate old leads from the last 90 days. Include one SMS, one email subject line, and one email body. Offer a reason to reply now without using fake urgency.

### Staff Training Script

Create a five-minute staff training script teaching a receptionist or admin how to handle new leads, missed calls, quote follow-ups, and review requests for [business type].

### Objection Handler

Generate responses to these common objections for [business type]: too expensive, need to ask spouse, found someone cheaper, not ready yet, and just researching. Keep each answer honest and helpful.

## Daily SOP

Run this once every business day.

1. Check every place leads arrive.
2. Move new inquiries into `New Lead`.
3. Reply to every `New Lead`.
4. Move priced inquiries into `Quoted`.
5. Follow up on quotes older than 24 hours.
6. Send review requests for completed jobs.
7. Tag every closed deal as `Won` or `Lost`.
8. Record why lost leads did not book.

## Weekly SOP

Run this every Friday.

1. Count new leads.
2. Count quotes sent.
3. Count booked jobs.
4. Count lost leads.
5. Read lost-lead reasons.
6. Improve one script or offer based on the most common lost reason.
7. Send one old-lead reactivation message to leads from 30-90 days ago.

## Automation Recipes

### Website Form to Follow-Up Sheet

Trigger: new website form submission.

Actions:

- Add row to Google Sheet.
- Send SMS or email reply using the speedy quote reply.
- Create reminder for 24-hour follow-up.
- Tag source as website.

### Missed Call to Text

Trigger: missed call from unknown or unsaved number.

Actions:

- Send missed-call text-back.
- Add contact to lead sheet.
- Create follow-up task for same day.

### Completed Job to Review Request

Trigger: job marked complete.

Actions:

- Wait two hours.
- Send review request.
- If no review after three days, send one gentle reminder.
